TUTORIAL-PRIMEIRO CONTATO COM O GITHUB

Criando uma conta no github
  • - Busque no seu navegador o site do git hub, coloque seus dados para criar uma conta nova
  • - Para criar um repositório clique em "create a new repository", depois nomeie o repositório em "repository name" e escolha se vai deixar privado ou público.
    • Configurações dentro da ide
  • Dentro da sua IDE (eclipse), já com o Git devidamente instalado, vá no canto superior esquerdo e clique na opção "window", na barra de pesquisa escreva git, logo você vai perceber que na opção " Default repository folder" o campo de texto está vazio, então clique em "browse", e será redirecionado para o seu explorador de arquivos, mais especificamente para a sua pasta de usuário, nela crie uma pasta nova com o nome de git para ser seu repositório. No eclipse, note que o campo vazio foi preenchido com o local da pasta, onde ficarão os seus projetos que serão clonados e enviados para o git hub. Agora é só clicar no botão de "apply and close".
    • Organizando o ambiente
  • - Para que o git apareça nas suas guias, siga novamente esses passos: window-> show view->other-> e digite git, clique em git staging e depois open, assim o ambiente irá aparecer juntamente com as informações dos estágios do seu projeto, fazendo esse mesmo processo adicione o git repositories e irá mostrar as suas pastas de repositório.
    • Enviando projeto
  • - Na plataforma do git hub crie um novo repositório com o nome desejado e com o token já gerado, volte para o eclipse para linkar essas duas plataformas
  • - Vá em window-> preference, procure por git e selecione configuration, para adicionar um novo user, vá em Add Entry, coloque seu nome de usuário e sua chave token.
  • - Agora para fazer o envio do projeto, clique com o botão direito em seu projeto, clique em team e depois em share project, na janela que aparecer, escolha create para decidir o repositório, por fim clique no botão finish duas vezes. Dessa forma, sua pasta será automaticamente criada, com o seu projeto vinculado à sua máquina, é dado o momento de enviar as modificações do projeto para o seu local remoto.
  • - Clique com o botão direito no seu projeto, vá em team -> add to Index, na barra de tarefas na área de " commit message" você pode deixar uma mensagem simbolizando o que foi feito, para enviar diretamente para a sua pasta e ao git hub, clique no botão " commit and push", na janela de configuração que aparece, coloque o endereço do seu git hub na URI e automaticamente seus dados serão preenchidos, clique em preview, depois em push que fará a ação do envio para o github.
  • - Depois, basta conferir no seu perfil do github na aba repositório e seu projeto estará lá.
